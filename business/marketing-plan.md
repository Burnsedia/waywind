# Waywind Marketing Plan

## Positioning
Use one theme source to keep your coding stack and desktop UI visually consistent.

## Core Channels
- GitHub and npm: discoverability, docs, release trust.
- X/Twitter: visual before/after demos and setup tips.
- Linux communities: ricing and Hyprland discussion channels.

## Content Cadence
- Weekly: one theming tip or preset example.
- Monthly: release notes with practical examples.
- Monthly: one short comparative demo clip.

## Funnel and CTAs
- Primary CTA: Install and run first theme generation.
- Secondary CTA: Sponsor project or request setup support.
- Events: npm_page_view, install_copy, first_generate, preset_use, repo_star, sponsor_click.

## 90-Day Experiments
- Test README quickstart variants with expected output examples.
- Test short video demos vs static screenshots on X.
- Test preset gallery page for install uplift.
